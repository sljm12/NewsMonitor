# Tech Stack: Global Pulse

## Backend
- **Framework**: FastAPI (Python)
- **RSS Parsing**: `feedparser`
- **Logic**: Rule-based entity extraction and keyword-based event categorization.
- **Environment**: Virtualenv with `uvicorn`.

## Frontend
- **Framework**: React 18+ with TypeScript
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Custom properties, Flexbox/Grid)
- **Theme**: Dark "Command Center" aesthetic with glassmorphism.

## Integration
- **API**: RESTful communication between FastAPI and React.
- **Configuration**: `.env` for environment variables.
